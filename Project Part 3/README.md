A full-stack web application for discovering, sharing, and managing movie reviews. Users can register, login, create reviews, edit or delete their own reviews, and browse reviews from other users. The app uses React on the frontend and Node.js/Express + MongoDB on the backend.

Features

User authentication (register, login, logout)

Role-based review management (create, read, update, delete only your own reviews)

Dynamic Navbar that updates based on login state

Responsive homepage and review pages

Fully connected frontend & backend via REST API

Tech Stack

Frontend: React, React Router, CSS Modules

Backend: Node.js, Express, MongoDB, Mongoose, bcryptjs

Authentication: JWT + localStorage
